\section{Externí datové a informační zdroje. Sekundární zdroje dat a informací, role competitive intelligence, přístup k externím zdrojům}

\subsection*{Primární a sekundární zdroje}
Materiály rozlišují primární a sekundární prameny informací. \term{Primární zdroj} přináší původní data nebo původní informaci. V analytice to mohou být přímo naměřená data, odpovědi respondentů, interní transakční záznamy, výsledky experimentu, logy nebo dokumenty vzniklé v organizaci.

\term{Sekundární zdroj} nepřináší původní data, ale pomáhá se zorientovat, kde primární informace hledat, nebo shrnuje již existující poznání. V materiálech jsou jako příklady sekundárních pramenů uvedeny slovníky, encyklopedie, lexikony, přehledové texty a informační zdroje sloužící k orientaci v tématu.

Pro datovou analytiku je dobré sekundární zdroje chápat šířeji: veřejné statistické databáze, open data portály, výroční zprávy, registry, akademické studie, oborové reporty, ceníky, katalogy, webové stránky institucí, dokumentace API a datové katalogy. Materiály zmiňují také externí data jako open data, tedy data, ke kterým lze získat přístup mimo firmu.

\subsection*{Ověřování kvality externích zdrojů}
U externích zdrojů je nutné ověřovat, zda jsou důvěryhodné a použitelné. Materiály doporučují ptát se zejména:
\begin{itemize}
  \item kdo je autorem zdroje a zda je autor dohledatelný;
  \item zda je autor spojen s relevantní institucí;
  \item zda jde o seriózní zdroj, například univerzitu, úřad, vědeckou instituci nebo odbornou organizaci;
  \item jak aktuální je stránka nebo datový soubor;
  \item zda je interpretace kompetentní a ověřitelná;
  \item zda text není příliš subjektivní a jednostranný;
  \item zda zdroj uvádí další odkazy, metodiku, metadata nebo dokumentaci.
\end{itemize}

V analytickém projektu se musí kontrolovat i datová kvalita: úplnost, soulad formátů, konzistence, přesnost, unikátnost, integrita, včasnost a validita. Externí data mohou mít jiné definice proměnných, jinou granularitu, jiný časový rozsah a jiný způsob sběru než interní data.

\subsection*{Přístup k externím zdrojům}
Přístup k externím zdrojům může mít různé podoby:
\begin{itemize}
  \item ruční stažení souboru, například CSV, XLSX, JSON nebo XML;
  \item dotazování přes API;
  \item přístup do databáze nebo datového katalogu;
  \item web scraping, pokud je právně a technicky přípustný;
  \item nákup dat od komerčního poskytovatele;
  \item využití veřejných dokumentů, výročních zpráv a registrů.
\end{itemize}

Materiály přímo nepopisují competitive intelligence, ale lze ji navázat na práci s externími zdroji. Competitive intelligence je systematické získávání, třídění, analýza a interpretace informací o trhu, konkurenci, zákaznících, dodavatelích, technologiích a regulatorním prostředí. Cílem není jen sběr dat, ale podpora rozhodování. Proto nestačí data stáhnout; je potřeba rozumět jejich původu, kvalitě, omezením a významu pro rozhodnutí.
